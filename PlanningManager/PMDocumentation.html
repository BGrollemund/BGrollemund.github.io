Planning

Créer :
Crée un nouveau planning vierge. Si vous voulez récupérer le planning en cours il restera disponible dans la zone de sauvegarde en bas de la page.

Modifier :
Il n'est possible de modifier que le nom du planning. Pensez à bien choisir les jours et la période lors de la création de votre planning.

Remarque :
Dans le soucis de respecter l'intégrité de l'Univers la date de fin ne peut être antérieure à la date de départ.

Activités et Intervenants

Ajouter :
Il est impossible d'ajouter deux activités (ou deux intervenants) avec le même nom.

Modifier :
Il n'y a pas de restrictions au niveau de la modification. Mettre le même nom aura pour effet de regrouper les heures dans les statistiques.

Supprimer :
Supprimer un élément n'affecte ni la planning, ni les statistiques. C'est une option permettant de réduire l'affichage. Il n'est alors plus possible d'ajouter des créneaux de ce type dans le planning. Pour pouvoir reprendre l'ajout il faudra recréer avec le même nom.

Créneaux

Ajouter :
Pour ajouter un créneau, cliquez sur le jour que vous souhaitez et remplissez les informations sur la gauche.

Modifier :
Le nombre d'heures et le type d'activités du premier créneau sont obligatoires. Sans ces informations le créneau restera ou redeviendra vierge.
De même, pour supprimer le second créneau renseignez une durée ou un type d'activités nul.

Mention :
Vous pouvez ajouter une information visible sur le planning concernant le créneau. Ceci n'a pas d'impact sur les statistiques.

Remarque :
Il n'est possible d'ajouter que deux créneaux par jour. De plus, la taille de ces créneaux ne dépend pas du nombre d'heures.

Demi-Groupes

Ajouter :
Sélectionnez le jour et le créneau où vous voulez ajouter faire deux demi-groupes, cliquez sur "Séparer" et remplissez les informations.

Statistiques et cas pathologiques :
1) Si les heures des deux groupes ne sont pas équilibrées, en type d'activités ou en intervenants, les statistiques affichées dans la section Activités (et ses détails) seront celles du groupe 1 (situé à gauche dans le planning).
2) Les heures en demi-groupe sont ajoutées à l'intervenant. Cependant, si l'intervenant réalise simultanément les cours des deux demi-groupes, les heures ne seront comptées qu'une fois. Dans ce cas, le détail des statistiques ne sera valable que pour les cours donnés au groupe 1.

Statistiques

Options Activités :
Vous pouvez créer des groupes et sous-groupes d'activités en utilisant le bouton "Ajouter" afin d'organiser et d'obtenir les sous-totaux.
Le bouton "Retirer" permet de retirer les activités cochées du groupe.
Le bouton "Supprimer" efface le groupe.

Options Intervenants :
En cochant les activités souhaitées et en validant par "Retirer" vous enlevez ces activités du calcul du total par intervenant. Les heures des activités retirées restent consultables dans le détail.

Sauvegarder et Charger

Sauvegarder :
Copier le texte de la première zone en bas de page, copier le dans le bloc-note , enregistrer le fichier txt et conserver le précieusement.

Charger :
Copier le texte du fichier txt, coller le dans la troisème zone en bas de page et cliquer sur "Charger".

Restauration :
Lorsque vous créez un nouveau planning, l'ancien est effacé. Il vous reste cependant une trace dans la deuxième zone en bas de page. Vous pouvez procéder comme pour charger un ancien planning et le restaurer.
